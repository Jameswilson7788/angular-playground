- 首先是MOCK，英文字典是假裝的意思，這邊的意思大概是我先假裝資料，我們剛剛設定的都是樣板格式這些東西，現在我們要假裝有資料進來，而MOCK就是一種假裝法，我們可以先記住格式，是一個陣列包住許多物件，物件都敘述著資料裡面的屬性。

以及所有資料顯現到前端上單純的<li>肯定是不夠顯現的所以要加上綴詞
```

```
